Hometown Homepage

This is the first real project in the Scrimba Frontend Developer Career Path. The challenge was to recreate a Figma design of a simple tourism web page.

Project description:

The primary purpose of doing this project was to learn how to use Figma and recreate this design. Figma was surprisingly easy to use, and the generated CSS was an excellent guide, especially for things like box-shadow, border-radius and line-height

I didn't make this website responsive because I tried to focus on recreating a static design as best as I could as a first exercise.

Built with

Figma, HTML5, CSS and Flexbox
