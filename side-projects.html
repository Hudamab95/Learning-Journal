Twimba (X "Twitter" Clone)

A frontend clone of X or Twitter.

Features:

Users can make a new tweet, users can read replies to a particular tweet, users can like and retweet, users can dislike, and unretweet, use of icons from Font Awesome CDN and generate uuid from UUID CDN.

Built with

Figma, HTML5, CSS, JavaScript, Fontawesome CDN and UUID CDN
